How to use

This Library Management System is software for monitoring and controlling the transection in a library. The project Library Management System is developed in MS Excel using VBA programming language, which mainly focuses on basic operations in a library of Motihari College of Engineering, Motihari. So, let’s start how to use this Library Management System efficiently. Click on the given tabs below and get it in details or Click here to read pdf file.

Visit Dark mode Screenshots click here.

Login

Step 1: Open Library_Management_System folder Double click on Library_Management_System.xlsm


Step2: Select Login on Login Panel.You will get different login buttons.


Click on Library Staff Login Button


Step3: Enter correct Username and Password and click on Login or double press enter key.


Students, Faculty and Admin staff can only see the Issue and Return books detail.


For Student, click on Student Loginand enter the username and password provided by college to get the Issue and Return book detail.


For faculty, click on Faculty Login and enter the username and password provided by college to get the Issue and Return book detail.


For admin staff, click on Admin Login and enter the username and password provided by college to get the Issue and Return book detail.

Add Books

Click on ADD BOOK tab or press Alt + B to open Book Entry Form. Fill the book details and click on ADD RECORD button or Press Alt + A .

Add Students Detail

Click on ADD STUDENT RECORD tab or press Alt + S to open Student Entry Form. Fill the book details and click on ADD RECORD button or Press Alt + A.

Add Faculty Detail

Click on ADD FACULTY DETAILS tab or press Alt + F to open Faculty Entry Form. Fill the book details and click on ADD RECORD button or Press Alt + A.

Issue Books

Click on ISSUE BOOK or press Alt + I to open Book Issue Form. Fill the book details and click on ISSUE BOOK button.

Return Books

Click on RETURN BOOK tab or press Alt + R to open Book Entry Form. Fill the book details and click on RETURN BOOK button.

Update Records


• To UPDATE BOOK RECORD: Type Book ID in BOOK ID textbox or in Book ID search box. The Book details will autofill if record found. Modify the data and click on UPDATE.


• To UPDATE STUDENT RECORD: Type Ref. No. in Ref. No. textbox or in Ref. No. search box. The Faculty details will autofill if record found. Modify the data and click on UPDATE.


• To UPDATE FACULTY RECORD: Type Roll No. in Roll No. textbox or in Roll No. search box. The Student details will autofill if record found. Modify the data and click on UPDATE.


Query Data

Click on Show Data Sheetbutton. Select "Query_Data" sheet and get the brief of Issue and Return book detail. For more detail visit HELP tab.

Library Rule

Open Library_Management_System folder and open LibraryRule folder and REPLACE the Library Rule image as LibraryRule in .jpg format.

Do not use .jpg while renaming image name. It’s an extension which is automatically included when we save a image in jpg format.


Follow this path to set Library Rule Image: E:\Library_Management_System\LibraryRule\LibraryRule.jpg

Help

Click on HELP tab or press Alt + H.

SHORTCUT KEYS

ADD RECORD Press Alt + A

ADD BOOK RECORD press Alt + B

ADD STUDENT RECORD press Alt + S

ADD FACULTY RECORD press Alt + F

ISSUE BOOK press Alt + I

RETURN BOOK press Alt + R

UPDATE RECORD Press Alt + A

LIBRARY RULE Press Alt + L

CLEAR / RESET FORM Press Alt + C

LOGOUT Press Alt + O

EXIT Press Alt + E

To Save the data, click on SAVE button.


To Show the data in Excel Sheet, click on SHOW DATA SHEET button and to hide the sheet click on HIDE DATA SHEET button.


To logout from userform, click on LOGOUT button or Press Alt + O


To change the theme to Dark Mode, click on Dark Mode button or for Light Mode, click on Light Mode button.


To close the program, click on EXIT button or Press Alt + E. All changes will saved at the time of closing program.
